# ikaros
伊卡洛斯是单体架构、好友模块去中心化的个人主页系统




# 天马行空计划功能列表总览
鸽.鸽..鸽...鸽....鸽.....鸽......鸽..........
- 个人数据存储计划
    - 个人数据上云计划
    - 图床计划
    - 数据云加密计划
- 个人日志记录管理展示中心
    - 最近动态(Trends Log)计划
    - 专栏(Column Log)计划
    - 视频生活(Video Log)计划
    - 音乐收听(Music Log)计划

- 音乐收藏计划
- P2P好友系统
- ...
- 其它未想出来的


# 技术栈
- 项目架构: 单体应用架构，好友系统去中心化分布式设计
- 开发语言: Java
- 后端框架: Spring, SpringMVC, SpringDataJpa, Hibernate
- 数据库: H2，MySQL
- 前端语言: HTML5, CSS3, JavaScript
- 前端框架: Vue.js, Vuetify
